# Hello World Project

This is a simple Node.js project that demonstrates a basic "Hello World" function.

## Description

The project contains a single file, `index.js`, which defines a function called `hello` that logs "Hello World" to the console. The function is called immediately when the script is run.

## Requirements

- [Node.js](https://nodejs.org/) installed on your machine.

## Usage

To run the script, use the following command in your terminal:

```bash
node index.js
```
